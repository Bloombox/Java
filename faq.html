General

What is Bloombox?

Bloombox is a company based in California that makes software for retail cannabis dispensaries. Our Java library allows integrating with Bloombox APIs from Java and Kotlin.

What can I do with Bloombox for Java?

Supported APIs by this client library include the Shop API, Menu API and Telemetry API. With the Shop service, you can do things like retrieve shop info, verify or enroll members, and submit and retrieve commercial orders. The Menu API allows retrieval of product catalog data. With the Telemetry API, developers can send their own telemetry events for later analysis.

How do I get a Bloombox account?

This library works with existing Bloombox accounts, and requires an API key. If you work for a dispensary and want to sign up, take a look at our website. If you're a developer or integrator and you want to build a product against Bloombox, drop us a line. We love APIs and share them freely! :)

Credentials

How do I get an API key?

There are a few ways you can get an API key:

  • Via the Bloombox Dashboard, in the Settings pane.
  • With an integrated Google Cloud project.
  • From Bloombox Helpdesk staff.
Where do I find my partner ID and location ID?

Your partner ID is a short string of text that you pick when you create your Bloombox organization account. You can find it in URLs related to your account, listed in the Settings pane of your Bloombox dashboard, or via the Helpdesk.
